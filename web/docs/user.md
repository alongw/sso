# 用户文档

关于授权，用户可自行选择允许客户端访问的范围，部分权限除外

### 权限对照表

| 名称     | 权限标识 | 默认值 | 是否必须 | 描述信息                                      |
| -------- | -------- | ------ | -------- | --------------------------------------------- |
| 基础信息 | 1        | true   | 是       | 读取用户基本信息（uid、用户状态、昵称、头像） |
| 电子邮箱 | 2        | true   | 否       | 读取用户电子邮箱地址                          |
|          |          |        |          |                                               |
